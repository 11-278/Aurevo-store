AUREVO E-Commerce Website

1. Project Overview
AUREVO is a complete frontend E-Commerce website project created for academic submission. The website design is inspired by Amazon and Daraz. The color theme is Black and Gold which gives it a modern and premium appearance. The project is fully responsive and works on mobile phones, tablets and desktop computers.

2. Main Features
Splash Screen: A 3 second loading screen appears when the website opens for the first time.
Product Categories: The website has 8 categories. Watches, Mobiles, Fashion, Beauty, Home Decor, Audio, Laptops and Sports.
Large Product Catalog: More than 50 products are added with real images, original prices, discount prices, ratings and number of reviews.
Product Details Page: When you click any product, a full details page opens with large images, description, specifications and reviews.
Shopping Cart: A cart page is included to show selected products, total price and checkout button.
Contact Page: A contact us form is added with name, email, subject and message fields.
Responsive Design: The layout adjusts automatically for all screen sizes.

3. Technology Stack
Frontend Only Project. No backend is used.
HTML5: For the structure of all pages.
CSS3: For custom styling and design.
Bootstrap 5.3: For grid system, navbar, cards and responsive layout.
JavaScript: For loading products from data file and cart functionality.
Font Awesome 6.4: For all icons used in the website.
Google Fonts: Poppins and Montserrat fonts are used.

6. How to Run This Project
Step 1: Download the complete AUREVO-Final folder to your computer.
Step 2: Open the folder in Visual Studio Code.
Step 3: Install the Live Server extension in VS Code.
Step 4: Right click on index.html file.
Step 5: Click on Open with Live Server.
Step 6: The website will open automatically in your default browser.
